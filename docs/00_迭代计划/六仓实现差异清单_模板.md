# 六仓实现差异清单（模板）

**用途**：用本表与 `blockchain-future` 组下六仓负责人对齐**「设计 SoT vs 实际实现」**的差异，是 [`docs/SoT_MAP.md`](../SoT_MAP.md) 与开发环境之间的桥梁。**每次对齐输出一份新文档**，命名 `_archive/diffs_YYYYMMDD.md`（即同步登记在 [`docs/00_迭代计划/设计变更记录.md`](设计变更记录.md)），保留作历史决策依据。

> **使用方式**：1) 复制本模板到 `_archive/diffs_<日期>.md`；2) 在与各仓负责人 1 小时对齐会前发出，会上逐行确认；3) 会后定稿，把"差异结论"回写到对应 SoT 文档。
