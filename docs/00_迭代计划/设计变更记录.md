## 登记规范

- **一行一变更**，不要把多条规则塞进同一行；摘要里只放结论，详细论证放 SoT 文档。  
- 涉及多条规则联动的，分多行登记，并在「摘要」相互引用。  
- **不要在本表辩论**；本表只读结论，讨论过程放 PR 描述或 ADR（见 [`adr/README.md`](adr/README.md) 启用后）。  
- **回滚**：若变更被推翻，新增一行「取代」类型并指向新版；不要修改历史行。
