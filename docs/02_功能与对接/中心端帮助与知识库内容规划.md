# 中心端帮助与知识库内容规划

本文档为中心运营端「知识库与帮助」页的完整内容规划，供产品/运营修订文案并与 [operator/help.html](../../operator/help.html) 同步更新。视角为**中心端运营人员**：配置任务、审核采集、调度 LCA、验收核查、报告下发与归档等。与 01、04、06、13、18 等业务文档一致。

## 一、快速开始

### 1.1 系统是什么

本系统为**钢铁行业产业链可信数据空间产品碳足迹数据服务系统**的中心运营端。您在此进行订单与任务管理、模板配置、采集审核、LCA 调度、核查验收、报告下发与归档，并管理供应商与核查机构。

### 1.2 您在本系统中的角色

您作为 **L3 中心运营人员**，核心工作包括：确认订单并分配任务、配置模板与快照、审核供应商填报数据（通过/驳回/提出疑点）、调度 LCA 计算、验收核查结果、在报告管理中下发报告、处理供应商申诉与归档上链。

### 1.3 典型操作路径（三步）

1. 在「订单管理」中确认已支付订单并**确认并分配任务**（自营/委托），在「任务管理」中进入各任务完成**配置**（选模板、下发）、**采集与审核**（通过/驳回）、**计算**（LCA）、**核查**（验收）。
2. 在「报告管理」中待三件套齐备后上传用印版、**勾选报告下发**至供应商，处理**申诉**（驳回/接受），供应商确认接收后执行**确认归档并上链**。
3. 在「供应商管理」「核查机构管理」中维护参与方信息与连接器状态；在「模板中心」维护采集模板。

### 1.4 入口与返回

侧栏可切换运营驾驶舱、订单管理、任务管理（自营/委托）、报告管理、供应商管理、核查机构管理、模板中心、结算、系统管理等；顶部可返回首页或退出。

## 二、业务与背景

### 2.1 产品碳足迹 (PCF) 与 LCA

- **产品碳足迹（PCF）**：产品全生命周期温室气体排放的量化与报告。
- **LCA（生命周期评价）**：由南钢 LCA 系统或运营侧完成计算；中心端负责将供应商采集的数据摆渡至 LCA、接收结果并进入核查阶段。
- **中心端职责**：配置任务与模板、审核采集数据、调度计算、验收核查、下发报告并归档，保障全流程合规与可追溯。

### 2.2 可信数据空间 (TDS) 与本系统

- 本系统运行在**钢铁行业产业链可信数据空间**上；中心端与行业功能节点（可信基座）协同，对接接入连接器。
- 供应商与核查机构通过连接器接入；您侧可见连接器状态、供应商证照（从空间同步）等，数据使用受数字合约与使用控制约束。

### 2.3 任务从哪来

订单由商城或外部系统产生，您在订单管理中确认并分配任务后生成子任务。任务经配置→采集→计算→核查→报告；报告在报告管理中独立管理：用印版上传、下发、供应商确认/申诉、归档上链。

## 四、流程与状态

### 4.1 任务五阶段

配置（您选模板、下发快照）→ 采集（供应商填报、您审核通过/驳回）→ 计算（LCA）→ 核查（第三方核查、您验收）→ 报告（在报告管理中下发、确认接收、申诉处理、归档）。中心端全程参与各阶段。

### 4.2 澄清与驳回（多阶段）

- **提出疑点/澄清**：采集、计算、核查阶段均可发起澄清；可转派给供应商，供应商在待澄清中回复。
- **驳回**：采集阶段驳回供应商提交→任务整改中；核查阶段验收驳回→责任方为核查机构；报告阶段驳回/接受供应商申诉。

### 4.3 报告阶段要点

下发前须上传用印版；仅供应商确认接收后可执行「确认归档并上链」；申诉中不可归档，须先处理申诉或等待确认接收。

### 4.4 采集阶段子状态

待提交、审核中、整改中、已完成。

## 五、安全与合规

### 5.1 数据使用控制

数据在系统内受「使用可信」约束；运营侧可见任务与报告全流程数据，用于审核、验收与归档；正式环境由策略引擎/连接器执行约束与审计。

### 5.2 供应商与核查机构数据

供应商证照与填报数据、核查报告与声明均来自可信数据空间或经连接器同步，请按合规要求使用与留存。

## 六、常见问题 FAQ

| 问题 | 答案要点 |
|------|----------|
| 任务配置后供应商看不到怎么办？ | 确认模板已选、快照已下发且任务处于采集阶段；检查供应商端连接器与权限。 |
| 采集审核时「提出疑点」和「驳回」如何选？ | 仅需说明→提出疑点；需改数或换凭证→驳回，任务变整改中。 |
| 报告何时可以下发？ | 三件套齐备（核查报告、核查声明、用印版核算报告）且已上传用印版后，在报告管理中勾选下发。 |
| 供应商申诉后如何处理？ | 在报告管理档案详情中「驳回申诉」或「接受申诉」；接受后可回退至核查等；申诉中不可归档。 |
| 归档上链的触发时机？ | 仅在点击「确认归档并上链」时触发；须供应商已确认接收且无待处理申诉。 |
| 核查机构任务从哪来？ | 任务进入核查阶段后，核查机构在其工作台可见待核查任务，与您侧任务 ID 一致。 |
