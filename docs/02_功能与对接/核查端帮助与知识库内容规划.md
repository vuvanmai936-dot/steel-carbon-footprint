# 核查端帮助与知识库内容规划

本文档为认证机构（核查机构）工作台「知识库与帮助」页的完整内容规划，供产品/运营修订文案并与 [certifier/help.html](../../certifier/help.html) 同步更新。视角为**核查机构（L4）**：接收待核查任务、穿透查看数据与凭证、通过/驳回/发起澄清。与 09、13、18 等文档一致。

---

## 一、快速开始

### 1.1 系统是什么

本系统为**钢铁行业产业链可信数据空间产品碳足迹数据服务系统**的认证机构端，即「第三方核查机构工作台」。您在此接收运营派发的待核查任务，查看供应商证照与填报数据、核算报告与凭证，并完成通过/驳回或发起澄清。

### 1.2 您在本系统中的角色

您作为 **L4 核查机构**，核心工作包括：在「核查任务列表」中领取/查看待核查任务，在任务详情中**穿透查看**供应商证照、工序数据与凭证、核算报告草稿，并执行**通过**、**驳回**或**发起澄清**（对报告草稿的澄清仅与运营可见；对供应商数据/凭证的疑问可经运营转派给供应商）。

### 1.3 典型操作路径（三步）

1. 在「核查仪表盘」或「核查任务列表」中查看待核查任务，点击进入**核查任务详情**。
2. 在详情页查看**供应商证照**（只读）、**核算报告与凭证**（只读），必要时对报告草稿或供应商数据发起**澄清**；完成核查后选择**通过**或**驳回**，并上传核查报告与核查声明证书。
3. 在「证书颁发管理」「结算中心」中查看证书记录与结算单；顶部「返回门户」可返回产品碳足迹应用门户。

### 1.4 入口与返回

侧栏可切换核查仪表盘、核查任务列表、证书颁发管理、结算中心、机构管理、知识库与帮助；顶部「返回门户」可返回产品碳足迹应用门户。

## 二、业务与背景

### 2.1 产品碳足迹 (PCF) 与核查

- **产品碳足迹（PCF）**：产品全生命周期温室气体排放的量化与报告。
- **第三方核查**：对运营侧提交的核算报告与支撑数据、凭证进行独立核查，出具核查报告与核查声明证书，保障结果可信。
- **核查机构职责**：在任务详情中穿透查看数据与凭证（只读）、对疑点发起澄清、完成通过/驳回并上传核查报告与声明证书。

### 2.2 可信数据空间 (TDS) 与本系统

- 本系统运行在**钢铁行业产业链可信数据空间**上；核查机构通过接入连接器与平台对接。
- 您查看的供应商证照、填报数据与凭证来源于可信数据空间或运营端同步数据，仅限在线查看，受使用控制约束。

### 2.3 任务从哪来

待核查任务由**中心运营端**在任务进入核查阶段后派发；任务编号（taskId）与运营端、供应商端一致，便于全链路追溯。

## 四、流程与状态

### 4.1 任务在核查阶段

任务由运营侧进入核查阶段后，您端可见该任务；您需在 SLA 内完成穿透查看、澄清（如需要）与通过/驳回，并上传**核查报告**与**核查声明证书**。上传完成后，该任务在任务管理侧结束，后续报告下发与归档在运营端报告管理中处理。

### 4.2 澄清的两类处理

- **核算报告草稿相关**：您对报告内容、结论、格式等有疑问时，发起澄清仅与**运营**可见，供应商不可见。
- **供应商数据/凭证相关**：您对供应商填报数据或凭证有疑问时，澄清先发往**运营**，运营确认后可选「自己回复」或「转发给供应商」，转发后供应商才可见并参与回复。

### 4.3 通过与驳回

**通过**：核查通过后上传核查报告与核查声明证书，任务进入报告阶段。**驳回**：验收不通过时选择驳回，责任方为核查机构，需补充或重新核查后由运营再次验收。

## 五、安全与合规

### 5.1 数据使用控制

您仅能**在线查看**供应商证照、填报数据与凭证、核算报告，受「使用可信」约束；禁止下载、截图、导出。正式环境由策略引擎/连接器执行。

### 5.2 证照与数据来源

供应商证照与任务数据来源于可信数据空间或运营端同步，仅供本次核查使用，请勿外泄或用于其他目的。

## 六、常见问题 FAQ

| 问题 | 答案要点 |
|------|----------|
| 待核查任务从哪里来？ | 由中心运营端在任务进入核查阶段后派发；任务 ID 与运营端、供应商端一致。 |
| 对报告草稿有疑问怎么办？ | 发起澄清，该澄清仅运营可见，您与运营沟通即可。 |
| 对供应商数据/凭证有疑问怎么办？ | 发起澄清，首接收方为运营；运营可转派给供应商后，供应商才可见并回复。 |
| 通过/驳回后还要做什么？ | 通过后须上传**核查报告**与**核查声明证书**，运营端据此进行报告下发与归档。 |
| 供应商证照从哪里来？ | 来自可信数据空间或运营端已同步数据，按任务关联供应商展示，只读。 |
