# 模板与标准中心逻辑说明

## 1. 业务目标
将非结构化的 Excel 文件转化为系统可识别的结构化数据配置（Schema），并定义供应商的数据提交流程。

## 2. Excel 解析逻辑 (前端/后端)
当用户上传 `.xlsx` 文件时，系统需执行以下解析：
1. **Sheet 识别**: 每个 Sheet 映射为一个“工序 (Process)”。
2. **Header 识别**: 默认读取第一行作为“字段名”。
3. **数据清洗**: 忽略空行、忽略隐藏 Sheet。

## 3. 凭证配置逻辑 (Evidence Configuration)
* **凭证与工作簿绑定**：凭证要求与整个工作簿（Workbook）绑定，不做行级关联。供应商需按凭证清单整体提交，不再配置「关联数据行」。
* **数据结构**（已废弃 `relatedFields`）:
  ```json
  {
    "requirementId": "req_001",
    "name": "电力发票",
    "description": "需盖章...",
    "exampleFileUrl": "http://..."
  }
  ```
* **废弃说明**：`relatedFields` 行级关联已废弃。原设计中可将凭证关联到具体数据行，现统一为与工作簿绑定；凭证清单由 Snapshot 的 `evidenceRequirements` 维护，通过 taskId 与任务关联。

## 4. Snapshot 格式规范

任务实例化时从模板复制出的快照，用于任务全生命周期，避免模板变更影响进行中任务。上传 Excel 后由 SpreadJS 解析并生成 `sheetData`。

```json
{
  "version": "1.0",
  "templateId": "TPL-xxx",
  "taskId": "TSK-xxx",
  "sheetData": {
    "M配料": [["类别","名称","单位","填报指引","数值","备注"], ["能源","电力消耗","kWh","请提供电费单","",""], ...],
    "M焙烧": [["类别","名称","单位","填报指引","数值","备注"], ["能源","天然气","m³","","",""], ...]
  },
  "evidenceRequirements": [
    { "id": "req_001", "name": "电力采购结算单", "desc": "需体现用电量及结算周期。" }
  ]
}
```

* **sheetData**：多 Sheet 二维数组，Key 为工序名（如 M配料、M焙烧），Value 为首行表头 + 数据行。供 SpreadJS `setArray` / `fromJSON` 使用。
* **evidenceRequirements**：凭证要求列表，与工作簿绑定，无行级关联。后续对接后端时 Snapshot 可存为 JSON/SSJSON，凭证文件单独存储，通过 taskId 关联。

## 5. SpreadJS 公共工具 (js/spreadUtils.js)

前端统一使用 **SpreadJS 16.x**（CDN：`gc.spread.sheets.excel2013white.16.0.0.css`、`gc.spread.sheets.all.16.0.0.min.js`）。`js/spreadUtils.js` 提供以下函数：

| 函数 | 作用 |
|------|------|
| `createSpreadFromSnapshot(hostEl, snapshot, options)` | 根据 Snapshot 创建 Spread，支持 `editableCols`、`readOnly` |
| `getSnapshotFromSpread(spread, extra)` | 导出当前 Spread 为 Snapshot 格式 |
| `setEditableColumns(spread, colIndexes)` | 指定可编辑列（如仅数值、备注） |
| `setSpreadReadOnly(spread)` | 全表只读 |
| `parseExcelFile(file)` | 解析上传的 xlsx，返回 sheetData（占位实现，需 SheetJS 或后端） |

### 各页 Spread 行为

| 页面 | 可编辑列 | 只读区域 | 凭证区 |
|------|----------|----------|--------|
| template_detail | 全表（草稿时） | 发布后整表锁定 | 右侧，与工作簿绑定 |
| task_detail_config | 名称、填报指引 | 表头、类别、单位 | 右侧，与工作簿绑定 |
| task_fill | 数值、备注 | 其余列 | 底部/右侧，与工作簿绑定 |
| task_detail_collect | 无 | 整表 | 右侧凭证审核 |
