# 核查机构端功能清单与入口说明

本文档描述核查机构端（L4 角色）的功能清单与入口。技术栈与供应商端一致：Vue 3、Element Plus、侧栏+顶栏+主区布局，详见 [js/certifierLayout.js](../js/certifierLayout.js)。

## 一、入口

- **门户入口**： [pcf.html](../pcf.html) 产品碳足迹认证服务页 → 点击「［L4］ 第三方认证机构」卡片，跳转至 **certifier/dashboard.html**（核查仪表盘）。
- **可选登录页**：certifier/login.html 为演示用登录（U-Key / 账号密码），登录成功后跳转 dashboard.html。

## 二、功能清单

| 序号 | 功能模块 | 说明 | 对应页面 | 优先级 |
|------|----------|------|----------|--------|
| 1 | 核查仪表盘 | 欢迎区、KPI 卡片（即将超时/新回复/待办总数/本周完成）、高优先级任务队列、个人效能趋势图、消息通知、常用标准库 | dashboard.html | P0 |
| 2 | 核查任务列表 | 待核查任务列表，穿透查看报告与凭证，操作：核查（跳详情） | task_list.html | P0 |
| 3 | 核查任务详情 | 任务与产品信息、核算报告与凭证（只读）、操作：通过/驳回/发起澄清 | task_detail.html | P0 |
| 4 | 证书颁发管理 | 证书记录列表、搜索、查看 | certificates.html | P0 |
| 5 | 结算中心 | 结算单列表、按单号/状态/日期筛选、确认结算、查看 | settlement.html | P1 |
| 6 | 机构管理 | 人员列表、添加成员、编辑 | admin.html | P1 |

## 三、菜单与路由

- **一级菜单**：核查仪表盘、核查任务列表、证书颁发管理、结算中心、机构管理。
- **详情页**：任务详情为列表跳转（task_detail.html?taskId=xxx），侧栏高亮「核查任务列表」。
- **布局**：el-container > el-aside(220/64) + el-header(60px) + el-main；侧栏标题「第三方核查机构」，主题色 #1e3a5f；头部面包屑 + 头像 + 返回门户（../pcf.html）。

## 四、与运营端/供应商端的关系

- 待核查任务与运营端任务管理中的「核查」阶段对应；核查机构完成通过/驳回/澄清后，运营端任务状态与报告管理衔接（参见 01、04、08 文档）。
- Mock 数据与 [07_Mock数据说明.md](07_Mock数据说明.md) 及运营端任务 ID、订单号保持一致，便于联调与演示。
