# SoT Map（Source of Truth 映射）

**用途**：每条业务规则只有**一份权威文档（SoT）**；其它文档/页面/开发仓只引用，不复述。本表是产品经理、开发与评审之间的「字典」。新人 5 分钟内必须看完此页。

> 同步约定：**新增/变更规则时，先改 SoT 文档，再在 [`00_迭代计划/设计变更记录.md`](00_迭代计划/设计变更记录.md) 登记一行，最后通知开发仓**。如未在本表登记，视为非现行规则。
